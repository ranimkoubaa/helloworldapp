# HelloWorld App — Java Maven

Projet réalisé dans le cadre du TP02 Git — Mastère DSIR1, ISET Sfax.

## Prérequis
- Java 17.0.12+
- Maven 3.9.9+
- Git 2.46.1+

## Installation
```bash
git clone https://github.com/ranimkoubaa/helloworldapp.git
cd helloworldapp
```

## Lancer le projet
```bash
mvn clean package
java -jar target/helloworldapp-1.0-SNAPSHOT.jar
```
